# 课表助手验收检查清单

## 1. V1.0 范围检查

- [ ] 产品只面向学生。
- [ ] 不需要登录账号即可使用。
- [ ] 课程数据保存在本地。
- [ ] 不支持云端同步。
- [ ] 不支持教务系统导入。
- [ ] 不支持课程提醒。
- [ ] 不支持单双周。
- [ ] 不支持多学期管理。
- [ ] 不支持课表分享。
- [ ] 不支持用户自定义课程颜色。

## 2. 课程管理

### 2.1 添加课程

- [ ] 用户可以进入添加课程页面。
- [ ] 用户可以填写课程名。
- [ ] 用户可以填写教师。
- [ ] 用户可以填写教室。
- [ ] 用户可以选择星期。
- [ ] 用户可以选择节次。
- [ ] 用户可以填写连续周次。
- [ ] 用户可以填写作业。
- [ ] 用户可以选择或填写考试日期。
- [ ] 用户可以填写备注。
- [ ] 保存成功后返回周视图。
- [ ] 新增课程显示在对应星期、节次和周次中。
- [ ] 新增课程自动分配颜色。

### 2.2 编辑课程

- [ ] 用户可以从课程详情页进入编辑页面。
- [ ] 编辑页面能回填原课程数据。
- [ ] 用户可以修改课程全部字段。
- [ ] 修改课程名后，周视图和详情页同步更新。
- [ ] 修改星期后，课程移动到新的星期位置。
- [ ] 修改节次后，课程移动到新的节次位置。
- [ ] 修改周次后，课程只在新周次范围展示。
- [ ] 编辑失败时保留用户已填写内容。

### 2.3 删除课程

- [ ] 用户可以从课程详情页触发删除。
- [ ] 删除前展示二次确认。
- [ ] 取消删除后课程仍存在。
- [ ] 确认删除后课程从周视图移除。
- [ ] 删除后不可再进入该课程详情页。

## 3. 数据校验

- [ ] 课程名为空时禁止保存。
- [ ] 课程名为空时提示“请输入课程名”。
- [ ] 星期未选择时禁止保存。
- [ ] 星期未选择时提示“请选择星期”。
- [ ] 节次未选择时禁止保存。
- [ ] 节次未选择时提示“请选择节次”。
- [ ] 周次为空时禁止保存。
- [ ] 周次格式错误时禁止保存。
- [ ] 周次错误时提示“请输入有效周次”。
- [ ] 起始节次大于结束节次时禁止保存。
- [ ] 起始周次大于结束周次时禁止保存。
- [ ] 教师为空时允许保存。
- [ ] 教室为空时允许保存。
- [ ] 作业为空时允许保存。
- [ ] 考试日期为空时允许保存。
- [ ] 备注为空时允许保存。

## 4. 周视图

- [ ] 默认进入后展示当前周课表。
- [ ] 页面展示当前周次。
- [ ] 页面展示周一至周日。
- [ ] 页面展示每天 12 节。
- [ ] 课程块展示课程名。
- [ ] 课程块展示教室。
- [ ] 课程块展示教师。
- [ ] 不同课程使用不同颜色。
- [ ] 同一课程在不同周次颜色保持一致。
- [ ] 当前周未上课的课程不显示。
- [ ] 当前周无课程时展示空状态。
- [ ] 点击课程块进入课程详情页。

## 5. 周次切换

- [ ] 用户可以切换上一周。
- [ ] 用户可以切换下一周。
- [ ] 切换周次后课程列表正确刷新。
- [ ] 课程只在 startWeek 到 endWeek 范围内展示。
- [ ] 不同周次切换时不会丢失课程数据。

## 6. 课程详情页

- [ ] 详情页展示课程名。
- [ ] 详情页展示教师。
- [ ] 详情页展示教室。
- [ ] 详情页展示星期。
- [ ] 详情页展示节次。
- [ ] 详情页展示周次。
- [ ] 详情页展示作业。
- [ ] 详情页展示考试日期。
- [ ] 详情页展示备注。
- [ ] 教师为空时展示“暂无教师信息”。
- [ ] 教室为空时展示“暂无教室信息”。
- [ ] 作业为空时展示“暂无作业”。
- [ ] 考试日期为空时展示“暂无考试日期”。
- [ ] 备注为空时展示“暂无备注”。
- [ ] 用户可以返回周视图。
- [ ] 用户可以进入编辑课程页面。
- [ ] 用户可以删除课程。

## 7. 课程冲突

- [ ] 同一星期、同一周次范围、节次重叠的课程识别为冲突。
- [ ] 不同星期的课程不识别为冲突。
- [ ] 同星期但周次不重叠的课程不识别为冲突。
- [ ] 同星期同周次但节次不重叠的课程不识别为冲突。
- [ ] 冲突课程允许保存。
- [ ] 保存冲突课程时给出明显提示。
- [ ] 周视图中冲突课程有可识别状态。

## 8. 本地数据

- [ ] 首次打开无数据时展示空课表。
- [ ] 新增课程后刷新页面，课程仍存在。
- [ ] 编辑课程后刷新页面，修改结果仍存在。
- [ ] 删除课程后刷新页面，课程不会恢复。
- [ ] 本地数据读取失败时页面不崩溃。

## 9. 响应式与体验

- [ ] 移动端主要流程可用。
- [ ] 桌面端周课表布局可用。
- [ ] 长课程名不会导致布局明显错乱。
- [ ] 长教室名不会导致布局明显错乱。
- [ ] 表单保存、取消、返回等操作反馈明确。
- [ ] 周视图切换响应时间主观上足够快，建议小于 300ms。

## 10. V1.1 上课地点导航

- [ ] 教室字段填写后可以触发地点解析。
- [ ] 地点解析成功后保存地点名称。
- [ ] 地点解析成功后保存地点经纬度。
- [ ] 地点解析失败时课程仍可保存。
- [ ] 地点解析失败时保留原始教室文本。
- [ ] 课程详情页在地点可用时展示导航入口。
- [ ] 教室为空时不展示可点击导航入口。
- [ ] 点击导航后打开高德地图。
- [ ] 高德地图以课程地点作为目的地。
- [ ] 用户拒绝定位权限时仍可打开目标地点。
- [ ] 高德地图 API 请求失败时不影响课程查看和编辑。
