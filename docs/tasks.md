# 课表助手开发任务拆解

## 1. 任务原则

- V1.0 先完成本地个人课表闭环。
- V1.1 上课地点导航只做数据结构和模块边界预留，不阻塞 V1.0。
- 每个任务完成后应能被独立验证。

## 2. 里程碑

| 里程碑 | 目标 | 范围 |
| --- | --- | --- |
| M1 | 基础工程与数据模型 | 项目结构、课程模型、本地存储 |
| M2 | 课程管理闭环 | 添加、编辑、删除、校验 |
| M3 | 周视图 | 当前周展示、周切换、课程块 |
| M4 | 课程详情 | 查看详情、空状态、删除确认 |
| M5 | 冲突与体验完善 | 冲突检测、颜色分配、边界状态 |
| M6 | V1.1 导航预留 | 地点字段、地图适配层接口 |

## 3. V1.0 开发任务

### T-001 工程初始化

- 确定技术栈和运行方式。
- 建立页面、组件、数据、工具函数目录。
- 增加基础样式和移动端适配约束。
- 建立本地开发启动方式。

验收：

- 项目可以本地启动。
- 首页可以正常渲染。

### T-002 课程数据模型

- 定义 Course 数据结构。
- 实现课程 ID 生成。
- 定义 weekday、period、week 的枚举或约束。
- 预留 V1.1 地点字段。

验收：

- 课程对象包含 PRD 需要的全部字段。
- 必填字段和可选字段边界清晰。

### T-003 本地存储

- 实现课程列表读取。
- 实现课程列表保存。
- 实现新增、更新、删除课程后的持久化。
- 处理首次进入无数据状态。
- 处理本地数据解析失败状态。

验收：

- 刷新页面后课程数据仍存在。
- 删除课程后刷新页面不会恢复。

### T-004 课程校验

- 校验课程名必填。
- 校验星期必选。
- 校验节次必选且范围为 1-12。
- 校验起始节次不大于结束节次。
- 校验周次必填且为连续周。
- 校验起始周次不大于结束周次。
- 校验考试日期格式。

验收：

- 必填项为空不能保存。
- 错误提示与 PRD 文案一致或语义一致。
- 非必填项为空可以保存。

### T-005 自动颜色分配

- 设计课程颜色池。
- 新增课程时自动分配颜色。
- 同一课程在不同周次保持同一颜色。
- 保证课程块文字可读。

验收：

- 多门课程颜色可区分。
- 编辑课程不改变原有颜色。

### T-006 添加课程

- 实现添加课程入口。
- 实现添加课程表单。
- 保存成功后返回周视图。
- 新课程显示在对应星期、节次和周次中。
- 保存失败或校验失败时保留用户输入。

验收：

- 用户可以成功添加课程。
- 添加后周视图立即展示新课程。

### T-007 编辑课程

- 从课程详情页进入编辑页。
- 表单回填原课程数据。
- 保存后更新本地数据。
- 修改星期、节次、周次后重新计算展示位置。
- 编辑失败时保留用户输入。

验收：

- 用户可以编辑课程任意字段。
- 编辑保存后周视图和详情页同步更新。

### T-008 删除课程

- 在课程详情页提供删除入口。
- 删除前展示二次确认。
- 取消删除后停留在详情页。
- 确认删除后移除课程并返回周视图。

验收：

- 删除后的课程不再出现在周视图。
- 删除后的课程详情不可继续访问。

### T-009 周视图页面

- 展示当前周次。
- 展示周一至周日。
- 展示第 1 节至第 12 节。
- 将课程块渲染到对应星期和节次。
- 当前周无课程时展示空状态。

验收：

- 当前周有课时课程位置正确。
- 当前周无课时空状态正确。

### T-010 周次切换

- 实现上一周按钮。
- 实现下一周按钮。
- 切换后重新过滤课程。
- 非当前周课程不展示。

验收：

- 课程只在 startWeek 到 endWeek 范围内展示。
- 切换周次后展示结果正确。

### T-011 课程详情页

- 点击课程块进入详情页。
- 展示课程名、教师、教室、星期、节次、周次。
- 展示作业、考试日期、备注。
- 展示字段空状态。
- 提供返回、编辑、删除入口。

验收：

- 详情页信息与课程数据一致。
- 空字段展示对应空状态文案。

### T-012 冲突检测

- 实现课程冲突判断。
- 添加课程时检测冲突。
- 编辑课程时检测冲突。
- 允许保存冲突课程。
- 给出明显冲突提示。
- 周视图中体现冲突状态。

验收：

- 同一天、同周次、节次重叠时识别为冲突。
- 不同周次或不同星期不误报冲突。

### T-013 响应式与可用性

- 优先优化移动端布局。
- 桌面端支持更宽周课表。
- 按钮、表单、课程块可点击区域足够明确。
- 长课程名、长教室名不破坏布局。

验收：

- 移动端和桌面端主要页面可正常使用。
- 文本不明显溢出或重叠。

### T-014 基础测试

- 测试课程校验。
- 测试周次过滤。
- 测试冲突检测。
- 测试新增、编辑、删除流程。
- 测试本地存储读写。

验收：

- 核心工具函数有单元测试或等效验证。
- 关键用户路径通过手动验收。

## 4. V1.1 上课地点导航任务

### T-101 地点字段接入

- 在课程模型中启用 locationName、locationAddress、longitude、latitude。
- 添加 / 编辑课程时支持保存地点解析结果。
- 地点解析失败时保留 classroom。

### T-102 高德地图适配层

- 封装地点解析 API。
- 封装导航 URL 构建。
- 处理 API 请求失败。
- 处理多个地点匹配。

### T-103 课程详情导航入口

- 地点解析成功时展示导航入口。
- 教室为空或地点不可用时不展示导航入口。
- 点击后打开高德地图路线规划或地点页面。

### T-104 定位权限处理

- 支持使用当前位置作为起点。
- 用户拒绝定位时只打开目的地。
- 不强制用户授权定位。

## 5. 建议开发顺序

1. T-001 工程初始化。
2. T-002 课程数据模型。
3. T-003 本地存储。
4. T-004 课程校验。
5. T-005 自动颜色分配。
6. T-006 添加课程。
7. T-009 周视图页面。
8. T-010 周次切换。
9. T-011 课程详情页。
10. T-007 编辑课程。
11. T-008 删除课程。
12. T-012 冲突检测。
13. T-013 响应式与可用性。
14. T-014 基础测试。
15. V1.1 再执行 T-101 至 T-104。
